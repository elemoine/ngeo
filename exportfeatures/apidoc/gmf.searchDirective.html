Class: searchDirective

searchDirective

new searchDirective() → {angular.Directive}

This file provides the "search" directive and controller for a GeoMapFish application.

FIXME: The 'placeholder' in the input field is hard-coded and can't currently be internationalised.

This directive uses the ngeoFeatureOverlayMgr to create a feature overlay for drawing features on the map. The application is responsible to initialize the ngeoFeatureOverlayMgr with the map.
